# 🚀 Campus Compass AI

An AI-powered web app that helps students generate personalized career roadmaps, study plans, and placement strategies.

---

## 💡 Features

* 🧠 AI-generated career roadmap
* 📅 Weekly study plan
* 🎯 Skill recommendations
* 💼 Project ideas for resume
* 📈 Placement preparation guidance

---

## 🛠 Tech Stack

* React.js
* Node.js
* Express.js
* Gemini AI
* REST APIs

---

## 🚀 Future Improvements

* Resume Analyzer
* Mock Interview Mode
* Learning Tracker

---

## ⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies
3. Add your API key in `.env`
4. Run frontend and backend
